使用

SVN地址:https://192.168.57.168/BrokerNet/VIPSTU/trunk/Sources/public/cloudUI

说明:Cloud-UI 是基于 Bootstrap 和 AngularUI 的 PC 端 UI 组件库，应用于云纪管理端业务。Cloud-UI 遵循云纪管理端统一的样式规范，有特定的 HTML 、 CSS 和 JavaScript 代码片段，使用时需严格遵循这些片段结构和规则。

目录结构

                        
                            └─cloudUI
                                ├─dist         ---> 生成目录
                                │  ├─css
                                │  └─fonts
                                ├─src          ---> 开发目录
                                │  ├─asset
                                │  ├─css
                                │  ├─fonts
                                │  └─less
                                └─docs         ---> 文档目录
                                    ├─css
                                    ├─js
                                    └─less

运用到项目中，确保项目已经引入以下文件:

                        
                            <!DOCTYPE html>
                                <html lang="zh-CN">
                                <head>
                                    <!-- Bootstrap 核心 CSS 文件 -->
                                    <link rel="stylesheet" href="css/bootstrap.min.css">
                                    <!-- Cloud-UI 核心 CSS 文件 -->
                                    <link rel="stylesheet" href="css/cloud-ui.css">
                                </head>
                                <body>
                                    <!-- Jquery 文件 -->
                                    <script type="text/javascript" src="js/jquery.min.js"></script>
                                    <!-- Bootstrap 核心 JS 文件 -->
                                    <script type="text/javascript" src="js/bootstrap.min.js"></script>
                                    <!-- Angular 核心 JS 文件 -->
                                    <script type="text/javascript" src="js/angular.min.js"></script>
                                    <!-- Angular-UI 核心 JS 文件-->
                                    <script type="text/javascript" src="js/ui-bootstrap-tpls-2.5.0.min.js"></script>
                                </body>
                            </html>

引入正确的文件后，在 module 中需要指定依赖的 module :

                        
                            angular.module('cloudApp',['ui.bootstrap'])
