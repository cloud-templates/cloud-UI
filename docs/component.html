Tab切换

Tab切换使用 Bootstrap的 Togglable tabs插件。

示例

  • 投顾管理
  • 角色管理
tab1的内容
tab2的内容

html结构

                        
                            <div>
                                <!-- 切换导航 -->
                                <ul id="myTab" class="tab-nav" role="tablist">
                                    <li class="tab-item active" role="presentation"><a class="tab-title" href="#tab1"
                                        aria-controls="tab1" role="tab" data-toggle="tab">投顾管理</a>
                                    </li>
                                    <li class="tab-item" role="presentation"><a class="tab-title" href="#tab2"
                                        aria-controls="tab2" role="tab" data-toggle="tab">角色管理</a>
                                    </li>
                                </ul>

                                <!-- 切换内容 -->
                                <div class="tab-content">
                                    <div id="tab1" class="tab-pane active" role="tabpanel">tab1的内容</div>
                                    <div id="tab2" class="tab-pane" role="tabpanel">tab2的内容</div>
                                </div>
                            </div>

js方法

每一个 Tab 都需要单独定义

                        
                            $('#myTab a').click(function (e) {
                                e.preventDefault()
                                $(this).tab('show')
                            })

模态框

不需写 JavaScript 代码也可激活模态框。给需要触发模态窗的元素（例如：按钮）添加 data-toggle="modal" 属性和 data-target="#myModal" 属性，就可以激活一个 id="myModal" 的模态框。

务必将模态框的 HTML 代码放在文档的最高层级内，也就是说，尽量作为 body 标签的直接子元素，以避免其他组件影响模态框的展现或功能。

此外，不支持同时打开多个模态框。千万不要在一个模态框上重叠另一个模态框。要想同时支持多个模态框，需要自己写额外的代码来实现。

样式

模态框基础 html 结构

                            
                                <!-- 点击触发模态框的按钮 -->
                                <button class="btn-normal bg-green" type="button" data-toggle="modal"
                                    data-target="#modalNormal">常规模态框</button>

                                <!-- 模态框基础结构 -->
                                <div class="modal fade cloud-ui-modal" id="modalNormal"
                                    tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
                                        <div class="modal-center">
                                            <div class="modal-dialog modal-normal">
                                                <div class="modal-content" role="document">
                                                    <!-- 模态框头部 -->
                                                    <div class="header">
                                                        <h2 class="modal-title" id="myModalLabel">新增组织机构</h2>
                                                        <i class="icon icon-close-circle" data-dismiss="modal"></i>
                                                    </div>

                                                    <!-- 模态框内容主体 -->
                                                    <div class="body">
                                                        <div>这里是内容</div>
                                                    </div>

                                                    <!-- 模态框底部按钮 -->
                                                    <div class="footer">
                                                        <div class="btn-box">
                                                            <button class="btn-big bg-green" type="button" data-dismiss="modal">保存</button>
                                                        </div>
                                                    </div>
                                                </div>
                                            </div>
                                        </div>
                                    </div>
                                </div>

在 .modal-dialog 类的基础上添加不同的类来定义不同大小的模态窗。

                        
                            <div class="modal fade cloud-ui-modal" id="modalNormal"
                            tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
                                <div class="modal-center">

                                    <!-- 常规模态框 -->
                                    <div class="modal-dialog modal-normal">
                                        <div class="modal-content"></div>
                                    </div>

                                    <!-- 小模态框 -->
                                    <div class="modal-dialog modal-small">
                                        <div class="modal-content"></div>
                                    </div>

                                    <!-- 大模态框 -->
                                    <div class="modal-dialog modal-big">
                                        <div class="modal-content"></div>
                                    </div>

                                    <!-- 自适应模态框 -->
                                    <div class="modal-dialog modal-adaptive">
                                        <div class="modal-content"></div>
                                    </div>
                                </div>
                            </div>

实现“点击并让弹出框消失”的效果需要一些额外的代码

为了更好的跨浏览器和跨平台效果，你必须使用 a 标签，不能使用 button 标签，并且，还必须包含 role="button" 和 tabindex 属性。

                            <div class="arrow"></div>
                            <div class="content">
                            ...
                            </div>
                            </div>

js方法

下拉框

下拉框插件统一使用 AngularUI的 Dropdown ，官方文档地址：https://github.com/angular-ui/bootstrap/tree/master/src/dropdown，参考文章地址：http://www.cnblogs.com/pilixiami/p/5636218.html。

示例

请选择

  • {{item}}

html结构

                        
                            <div ng-controller="DropdownController">
                                <div class="dropdown-box" uib-dropdown>
                                    <div class="select-box" uib-dropdown-toggle>
                                        <i class="triangle-downward"></i>
                                        <p class="select-text js-select-text">请选择</p>
                                    </div>
                                    <ul class="dropdown-menu" uib-dropdown-menu>
                                        <li class="select-item" ng-repeat="item in items" ng-click="chooseItem(this)" uib-dropdown-menu>
                                            <p class="select-item-text" title="{{item}}">{{item}}</p>
                                        </li>
                                    </ul>
                                </div>
                            </div>

Dropdown 中可以使用的属性有：

属性名 默认值 备注
auto-close always

可设置的值有三个：

always：点击菜单后自动关闭。

disabled：不自动关闭，使用is-open手动控制。

outsideClick：点击Dropdown外部的元素时关闭。
dropdown-append-to null 将菜单项（ul部分）放在其他DOM元素中，而不是包含在uib-dropdown所在的元素中。
dropdown-append-to-body false 将菜单项（ul部分）放在body末尾，而不是包含在uib-dropdown所在的元素中。
is-open false 菜单是否展开。uib-dropdown-toggle所在的元素点击时会更改这个值。
keyboard-nav false 是否允许方向键选择菜单。
on-toggle 菜单展开或收起时触发的方法。

js方法

在 controller 中定义下拉列表的内容：

                        
                            app.controller('DropdownController',function($scope) {
                            $(scope).items = [
                                '第一项',

                                '第三项第三项第三项第三项第三项第三项第三项第三项第三项第三项第三项',
                                '第四项',
                                '第五项',
                                ]
                            })

分页

分页插件统一使用 AngularUI的 Pagination，官方文档地址：https://github.com/angular-ui/bootstrap/tree/master/src/pagination，参考文章地址：http://www.cnblogs.com/pilixiami/p/5634405.html。

示例

    html结构

                            
                            <div class="page-container" ng-controller="PaginationController">
                                <ul uib-pagination ng-model="currentPage" total-items="totalItems" max-size="maxSize" previous-text="上一页"
                                    next-text="下一页" boundary-links="false" boundary-link-numbers="true">
                                </ul>
                            </div>

    Pagination中可以使用的属性有：

    属性名 默认值 备注
    boundary-links false 是否显示第一页和最后一页的按钮
    boundary-link-numbers false 是否显示第一页和最后一页的页数,并在页数过多时用…表示被隐藏的页数
    direction-links true 是否显示上一页和下一页的按钮
    first-text first 第一页的按钮的名字
    last-text last 最后一页的按钮名字
    previous-text Previous 上一页的按钮名字
    next-text Next 下一页的按钮名字
    force-ellipses false 是否在rotate被设置为true并且页数过多时显示为"…"
    rotate true 是否保持当前在可视范围的中间
    items-per-page 10 每页显示的数量.设置值小于1表示显示所有项
    max-size null 可选择的页数范围(如果设置为5,当前页为10,总页数为100,那么可选择第8,9,10,11,12页)
    ng-change 页数变化时调用的函数
    ng-disabled false 是否禁用
    ng-model 当前页数
    num-pages angular.noop 只读属性，表示总页数
    page-label angular.identity 设置页数标签的表达式
    total-items 总共有多少条数据

    日历插件统一使用 AngularJs的UI组件 UI-Bootstrap中的 Datepicker Popup，官方文档地址：https://github.com/angular-ui/bootstrap/tree/master/src/datepickerPopup，参考文章地址：http://www.cnblogs.com/pilixiami/p/5611346.html。

    示例

    html结构

    Datepicker Popup 中可以使用的属性有：

    属性 默认值 备注
    alt-input-formats [] 手动输入日期时可接受的格式
    clear-text clear 清空按钮的文本
    close-on-date-selection true 选择一个日期后是否关闭日期面板
    close-text close 关闭按钮的文本
    current-text today 今天按钮的文本
    datepicker-append-to-body false 是否将日期面板放在body元素中
    datepicker-options 对Datepicker控件的配置
    is-open false 是否显示日期面板
    on-open-focus true 打开日期面板时是否获取焦点
    show-button-bar true 是否在日期面板下方显示”今天”,”关闭”等按钮
    type text 可以改为date|datetime-local|month，这样会改变日期面板的日期格式化。
    popup-placement auto bottom-left

    在位置前加一个auto,表示日期面板会定位在它最近一个可以滚动的父元素中.可以设置的位置有:

    top top-left  top-right  bottom  bottom-left  bottom-right  left  left-top   left-bottom  right  right-top  right-bottom
